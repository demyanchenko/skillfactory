В описании (README) кратко укажите:

Задача проекта:
применить на практике инструменты Python для проведения аналитики, для получения конкретных ответов на основе анализа большого объёма данных.

Количество набранных в викторине баллов:
не знаю!

Ответы на вопросы саморефлексии:

1. Какой частью своей работы вы остались особенно довольны?
Мне особо понравилось создание дополнительных таблиц с сохранёными данными из множественных полей основной таблицы. Этот подход был переломным моментом, позволившему быстро решить все последующие задачи.

2. Что не получилось сделать так, как хотелось? Над чем ещё стоит поработать?
Не удалось применить сводные таблицы для одного из вариантов решения (в итоге решил задачу другим методом).
Надо ещё понять, какие задачи можно порешать с помощью сводных таблиц.

3. Что интересного и полезного вы узнали в этом модуле?
Узнал, как вариативно можно решить одну задачу. Не всегда существует один единственно правильный вариант - их может быть несколько.

4. Что является вашим главным результатом при прохождении этого проекта?
Овладел техникой работы с DataFrame.

5. Планируете ли вы менять стратегию изучения последующих модулей?
Да, теперь больше практики буду делать, искать новые варианты решения.
